# 📝 Advanced To-Do List

A modern To-Do List web application built using HTML, CSS and JavaScript.

## 🚀 Features

- ➕ Add Tasks
- ✏️ Edit Tasks
- ❌ Delete Tasks
- ✅ Mark Tasks as Completed
- 🔍 Search Tasks
- 📂 Filter (All / Pending / Completed)
- 🌙 Dark Mode
- 📊 Progress Bar
- 💾 Local Storage
- 🗑️ Clear All Tasks
- ⌨️ Enter Key Support

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
